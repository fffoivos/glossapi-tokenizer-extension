Β.
ii · method

What was measured.

A document-level language label, tokenised through the Apertus encoder, summed per vocab id. Raw counts only — no derived attribution, no posterior, no argmax.
strategy

Trust the source dataset's per-language partition as the language label (FineWeb-2 uses GlotLID v3, the same label Apertus's pipeline trusted). Stream documents, tokenise with the Apertus tokeniser, increment a per-language int64 histogram of length 131 072 via np.bincount. Stop when the per-language cumulative reaches 10⁹ tokens.

apparatus

Workers w0–w7 partition the 1 933 canonical keys by SHA-1 modulo 8, giving 208–263 keys per worker. Sources used, in priority: FineWeb-2-HQ → FineWeb-2 → Clean-Wikipedia → EuroParl → ParaDocs → FineWeb-Edu → FineWeb-HQ → DCLM-Edu. Pre-computed file listings (31 750 paths, 1.25 MB JSON) shipped with each worker to avoid the 1 000 req / 5 min Hugging Face rate limit.

tok.encode_batch(buf, add_special_tokens=False) → np.bincount(ids, minlength=131072)

Final artefacts under outputs/: a (1933 × 131072) int64 matrix, per-key metadata, and a 131 072-row token-features table with twenty script flags. Deliberately not produced: primary-language, confidence, top-K signature, entropy. Those belong downstream, after we have looked.

x · three quiet tokens

The Greek-script entries that did not fire.

Of 1,507 vocab entries carrying at least one Greek codepoint, three did not appear in a billion tokens of Greek text. All three are the same shape — the lowercase μ used as a measurement prefix.
▁μ
id 49,669 · "▁μ"
The Greek letter μ, word-initial, on its own. Used in scientific writing as the prefix "micro-". FineWeb-2-HQ's quality filter discards drug-info and dosage text where this would fire.
▁μg
id 121,271 · "▁μg"
μ followed by lowercase g — "micrograms". Again, a pharmaceutical-text token absent from quality-filtered Greek web prose.
▁μg
id 122,838 · "▁μg" variant
A second byte-fragment merge of μg. The tokenizer has two entries; both miss for the same reason.

These tokens are not waste. They would fire on technical Greek text — medical leaflets, lab reports, dosage labels — that the FineWeb-2-HQ pipeline filters out. The vocabulary inherited from Mistral-Nemo is broader than the corpus Apertus consumed.

Reading order

This page is a first-pass reading. The raw matrix and metadata are intentionally lossless; primary-language, signature, and confidence are decisions to be made downstream, after these patterns have been seen.
